#### 题意
给定一个double类型的浮点数base和int类型的整数exponent。求base的exponent次方。
#### 题解
可用快速幂来降低复杂度。注意指数为负的情况。
```cpp
class Solution {
public:
    double Power(double base, int exponent) {
        double ans = 1.0;
        bool positive = exponent>=0;
        exponent = abs(exponent);
        while(exponent){
            if(exponent&1){
                ans*=base;
            }
            base*=base;
            exponent>>=1;
        }
        if(!positive) return 1.0/ans;
        return ans;
    }
};
```
